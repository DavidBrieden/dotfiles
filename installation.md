# Installation Guide

## Overall
Clone this repo in your home folder and execute install.sh. This will create some symlinks in your .config folder.

## ZSH
install
 - zsh
 - antigen (probably `curl -L git.io/antigen > antigen.zsh`)
The .zshrc should contain everything you need. Extra Plugins will be downloaded via the antigen package manager.
Make sure zsh is your default shell (depending on your OS).
